# MUSICA 🎶

An AI-powered music recommender that suggests songs based on your current mood.

## Live Demo
[musicaa.streamlit.app](https://musicaa.streamlit.app)

## Run Locally
git clone https://github.com/smaranhere/Musica
cd Musica
pip install -r requirements.txt
streamlit run app.py

Add your GROQ_API_KEY to a .env file before running.

## Tech Stack
- Python
- Streamlit
- Groq API
- LLaMA 3.1
